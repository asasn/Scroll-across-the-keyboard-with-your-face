## 数据相关和文件格式

>本软件具有简单的导入导出功能，可以将章节保存为`.txt`格式文件

>书籍数据保存在软件`/books`目录下的相关文件里，数据库框架为`sqlite`

>数据库文件为`.db`类型文件，书籍封面为同名`.jpg`文件

>建议配合[坚果云](https://www.jianguoyun.com/)同步盘使用，可以把`/books`文件夹整个同步
